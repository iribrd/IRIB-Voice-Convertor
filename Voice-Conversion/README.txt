For Training:

Step 1. Generate spectrogram data from the wav files: 
python make_spect.py

Step 2. Generate training metadata, including the GE2E speaker embedding (please use one-hot embeddings if you are not doing zero-shot conversion):
python make_metadata.py 

Step 3. Run the main training script
python main.pyÂ 

Converges when loss is around 0.0001


For Testing:

Step 1. Generate spectrogram data from the wav test files: python make_spect.py


Step 2. Generate testing metadata:(please changes the input folder)
python make_metadata.py
or
python make_metadata_test.py


Step 3. Generate Output by:
python convert.py(if test metadata file is generated by make_metadata_test.py)
or
python convert20.py(if test metadata file is generated by make_metadata.py)
